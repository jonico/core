Developer documentation for the CCF
If you read through all the wiring files of the predefined example scenarios and had a look into our glossary of terms, you should be able to understand the source code of the CCF as well. The source code can be found within our SVN repository which can be browsed online as well as the generated javadocs. If you have any questions regarding the code, discovered a bug, like to contribute a new reader or writer component or just want to get in touch with us, just join our developer mailing list.
