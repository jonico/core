Current Release: 1.0 - June 2008

Initial framework built on openAdaptor.  Contains read/write support for SFEE Tracker Artifacts (version 4.4 SP1 and higher).  Support for these example scenarios .

Next Release: 1.1 - July/August 2008

Add support for CEE Project Tracker artifacts (version 5.1 and higher).  Stabilization and improvements in runtime performance and memory usage.

Future Releases: In planning

We plan to investigate graphical or web browser interfaces to simplify the configuration process as well as provide easier access to the logging information and monitoring runtime behavior.
