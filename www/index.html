CollabNet Connector Framework

This is the home of the CollabNet Connector Framework (CCF) a.k.a Generic Conenctor and Connector SDK. The CCF is built on top of openAdaptor and allows you to easily build connectors with CollabNet Project Tracker and Tracker.

CollabNet Connector Framework is an generic framework that simplifies application integration between your systems.

Integrating between two systems is as simple as:
0. Download and install CCF
1. Specify your environment: What the systems available in your environment
2. Configure the connectivity: Specify which instance of the system should map to which instance of the
other system. Provide the authentication information and the field mappings.
3. Start CCF and watch the fun!

Currently supported systems: As of v0.1, CCF can support connectivity among the following systems:
- HP Quality Center (versions 9.0, 9.2)
- Collabnet SFEE (versions x.x)
- Collabnet Project Tracker (CEE 5.0)
(You are welcome to provide subsystems for other products. Please refer to the 'How to Contribute' section
below)
