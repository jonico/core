								Generic Artifact Template
								=========================
								
This file describes aspects relating to the Generic Artifact and its template.

1. Specification:
	The specification for the generic artifact is provided at:
	http://ccf.open.collab.net/source/browse/*checkout*/ccf/trunk/docs/Generic%20Artifact%20Template.doc
	An XML schema is also available at http://ccf.open.collab.net/source/browse/*checkout*/ccf/trunk/src/core/com/collabnet/ccf/core/ga/genericartifactschema.xsd?revision=51
	
2. Helpers:
	The GenericArtifactHelper class contains methods to create a Generic Artifact. All Reader classes
	are expected to use this class to generate generic artifacts.
